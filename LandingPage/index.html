JavaScript

Es un lenguaje de programación o de secuencias de comandos que te permite implementar funciones complejas en páginas web.

Html 5

Es un lenguaje de marcación de elementos para la creación de documentos hipertexto, sus siglas que corresponde a HyperText Markup Language.

Visual Studio

Visual Studio es un entorno de desarrollo integrado para Windows y macOS. Es compatible con múltiples lenguajes de programación, tales como C++ y C#.

Java

Entorno de computación originaria de Sun Microsystems, capaz de ejecutar aplicaciones desarrolladas en dicho lenguaje de programación.
